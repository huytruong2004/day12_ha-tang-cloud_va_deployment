# Deployment Information

## Public URL

https://ai-agent-production-baxc.onrender.com

## Platform

Render (Blueprint, Docker runtime, free plan, Singapore region)

## Configuration

Deployed from `render.yaml` (repo root) which pulls from the `06-lab-complete/` subdirectory.

## Test Commands

### Health check

```bash
curl https://ai-agent-production-baxc.onrender.com/health
# Expected: 200 OK with JSON {"status":"ok", "uptime_seconds":..., ...}
```

### Readiness check

```bash
curl https://ai-agent-production-baxc.onrender.com/ready
# Expected: 200 OK {"ready": true}
```

### Authentication required

```bash
# Without API key → 401
curl -X POST https://ai-agent-production-baxc.onrender.com/ask \
  -H "Content-Type: application/json" \
  -d '{"question": "Hello"}'
# Expected: 401 {"detail":"Invalid or missing API key..."}
```

### Valid request

```bash
curl -X POST https://ai-agent-production-baxc.onrender.com/ask \
  -H "X-API-Key: YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"question": "What is deployment?"}'
# Expected: 200 OK with AskResponse payload
```

### Rate limiting (20 req/min per `RATE_LIMIT_PER_MINUTE`)

```bash
for i in $(seq 1 25); do
  curl -s -o /dev/null -w "%{http_code}\n" \
    -X POST https://ai-agent-production-baxc.onrender.com/ask \
    -H "X-API-Key: YOUR_KEY" \
    -H "Content-Type: application/json" \
    -d '{"question":"test"}'
done
# Expected: 200 for first 20, then 429 for the rest
```

## Environment Variables

Set via `render.yaml`:

- `ENVIRONMENT=production`
- `APP_VERSION=1.0.0`
- `OPENAI_API_KEY` (optional — unset → uses mock LLM)
- `AGENT_API_KEY` (auto-generated by Render)
- `JWT_SECRET` (auto-generated by Render)
- `DAILY_BUDGET_USD=5.0`
- `RATE_LIMIT_PER_MINUTE=20`
- `PORT` (auto-injected by Render)

Retrieve the live `AGENT_API_KEY` from the Render dashboard → Service → Environment tab.
